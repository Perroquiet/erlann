ERLANN is an Artificial Neural Network (ANN) Library using the Erlang programming language.

Specifications:

The library currently uses a single Erlang process for a single perceptron (neuron).
  • Data Preprocessing: Log Normalization Function
  • Activation Function: Sigmoid Function
  • Learning Algorithm: Backpropagation
  • Training Sequence: Training -> Testing -> Error evaluation
  • Evaluation Criteria: Mean Squared Prediction Error (MSPE)
